**Mac OS 快捷键记录**

-   Command+C：复制
-   Command+V：粘贴
-   Command+Option+V：剪切（移动）
-   Command (⌘) + M: 最小化窗口
-   Command (⌘) + W: 关闭窗口
-   Command (⌘) + Q: 关闭应用程序（退出）
-   Option + 鼠标左键文件：快速复制
-   Option 并在任意选中的文件夹上双击：文件夹多开
-   调度中心 Option：可显示 X
-   Command-空格：唤起全局搜索
-   Alt-Tab ：快速切换前台应用
-   Ctrl-⬆️：打开调度中心，可点击右上角加号，新建虚拟桌面
-   Ctrl-⬅️/➡️：切换虚拟桌面
-   Control + Command (⌘) + F： 全屏(退出)

**Chrome**
