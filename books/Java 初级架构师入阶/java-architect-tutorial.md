该系列只是对自己在 java 架构方面的现阶段理解，随着时间流逝，会持续校正。

写这个的目的不是给别人看，只是记录总结一下自己这方面的思考。

**为什么要按一个系列来写**？

如果你有幸看到这个，希望能收到你的见解，互相交流学习。
